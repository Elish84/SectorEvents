# Workflow: Standard Delivery Cycle

## 1. Intake
User sends a request to Project Manager.

## 2. Decomposition
Project Manager creates:
- request_id
- one or more tasks
- dependencies
- success criteria

## 3. Delegation
Project Manager routes tasks to specialist agents.

## 4. Specialist Execution
Each agent returns a structured response.

## 5. Synthesis
Project Manager:
- checks consistency
- resolves conflicts
- approves or requests follow-up
- prepares one final answer for the user

## 6. Memory Update
Summarizer updates project memory.

## 7. Next Cycle
Project Manager uses the updated memory as context for future requests.
